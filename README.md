# hello bun



Hello is an notion style ai based language learning app written using the following technologies:

- Next   (Framework)
- Bun    
- OpenAI (Research)
- Chadcn (Base UI Components)
- Radixb (Accessibility)


## Getting Started

1. install dependencies

```
bun install
```

2. run the development server:

```bash
bun run dev
```
